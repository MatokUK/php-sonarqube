Logical expression is always true / false.

Expression is poorly written or is altered because of debugging. Expression contains true (expression implicitly converted to true) with logical or || are always true and expressions contains false (expression implicitly converted to false) with logical and && are always false.

Noncompliant Code Example

    if (1 || $isDevEnvironment) { // Noncompliant
    }

    if (true || $isDevEnvironment) { // Noncompliant
    }

    if (0 && $recurrentPayment) { // Noncompliant
    }

    if (false && $isDevEnvironment) { // Noncompliant
    }
